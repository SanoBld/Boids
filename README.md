# \# Boids — Flocking Simulation

# 

# A minimalist, high-performance biological flocking simulation based on Craig Reynolds' 1986 "Boids" algorithm. This project is a functional \*\*experiment\*\* in emergent behavior, exploring how simple local rules create complex global patterns.

# \## The Experiment

# 

# This simulation is designed as an interactive playground to observe how three basic steering behaviors interact in real-time:

# 

# 1\.  \*\*Separation:\*\* Steering to avoid crowding local flockmates.

# 2\.  \*\*Alignment:\*\* Steering towards the average heading of local flockmates.

# 3\.  \*\*Cohesion:\*\* Steering toward the average position (center of mass) of local flockmates.

# 

# By adjusting the weights of these forces, you can observe transitions between chaotic movement, structured schooling, and organic-like "flocking" flows.

# 

# \## Features

# 

# \-   \*\*Real-time Interaction:\*\* Tweak forces (Cohesion, Alignment, Separation) on the fly via the sidebar.

# \-   \*\*Visual Debugging:\*\* Toggle visibility for perception radii to see how each boid "perceives" its environment.

# \-   \*\*Live Metrics:\*\* Real-time tracking of boid counts and spatial data.

# \-   \*\*Customization:\*\* Multiple boid designs (Arrows, Circles, Dots) and color themes (Cosmic, Mint, Ember, Mono).

# \-   \*\*Optimized Performance:\*\* Built with vanilla JavaScript and HTML5 Canvas for smooth 60FPS animation.

#
